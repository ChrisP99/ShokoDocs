The following is a compilation of known programs and plugins that utilize Shoko.

If you are aware of a program that uses Shoko Server or are currently working on one yourself, you can contribute to
the list by clicking the **Edit This Page** button on the right.

:::note[About Maintenance Mode & Legacy Mode]
A program or plugin marked as **Maintenance Mode** indicates a halt in active development and support, primarily due to
the substantial time and effort needed for its upkeep. If a program or plugin is in **Legacy Mode**, it is incompatible
with the current version of Shoko Server; see the Legacy table for more information.
:::
